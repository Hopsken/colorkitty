## Feature

1. Extract color palettes from pictures.
2. Specify the number of colors you want.
3. Drag to arrange color's order.
4. Show related colors to help generate palettes.
5. Export final palette as PNG file.
6. More...

## Start

```bash
yarn && yarn start
```
